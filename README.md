## Description du site

Site web du cabinet de psychologie d'**Emma Rouche**, psychologue clinicienne a Castelnau-le-Lez (34170), en proximite de Montpellier.

### Adresse

34170 Castelnau-le-Lez

### Services proposes

- **Consultations adultes** - Psychotherapies pour adultes
- **Consultations enfants et adolescents** - Psychotherapies pour les jeunes
- **Soutien parental et familial** - Accompagnement des parents et familles
- **Bilans psychologiques** - Evaluations et bilans complets

## Methode de recuperation

Les fichiers ont ete telecharges via l'API de la Wayback Machine (Internet Archive) en utilisant le modificateur `if_` pour obtenir le contenu original sans les elements d'interface de l'archive.

Outil utilise : [waybackpack](https://github.com/jsvine/waybackpack) + curl sur l'API CDX d'Internet Archive.
